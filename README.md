# HML - a HyperCubesPlanet modloader

> [!NOTE]
> The 2.0 update is still in beta. All bugs should be reported in this project's issues section.

## How to download
> [!IMPORTANT]
> We only support browser for now.
1. Download Tampermonkey on your browser
2. Install [the userscript](https://raw.githubusercontent.com/ximi-ovh/HML/main/modloader.user.js)
3. Open [the game](https://hypercubesplanet.web.app) in your browser. A window should show up after loading.
